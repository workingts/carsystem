# CAR SYSTEM - Handoff Edition

> ⚠️ 본 버전은 핸드오프(인계) 목적으로 제공된 기초 버전입니다.
> 추가 개발 및 완성은 수령자의 책임입니다.

---

## 개요

차량검사 접수 관리 시스템 기초 버전입니다.
MIT 오픈소스(Rent_Auto_Desktop)를 기반으로 제작되었습니다.

---

## 기능

```
✅ 4륜 차량 검사 접수
✅ 이륜 차량 검사 접수
✅ 접수 목록 조회 (개인정보 마스킹)
✅ 검사대상 차량 자동 리스트 (D-90~D+31)
✅ 수수료 자동 계산 (TS 공단 기준)
✅ CSV 내보내기
✅ 집계표
✅ 개인정보 AES-CBC 암호화
```
